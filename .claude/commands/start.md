---
description: Use at the start of a work session to establish a clear goal. Prompts for what "done" looks like and holds the session accountable to it.
---

# Session Start

Ask the user this exact question — nothing else first:

> **What does done look like for this session?** Describe the specific thing you want committed and working by the end. One sentence is enough.

After they answer:
1. Confirm the goal back in one sentence: "Got it — we're done when [their goal]. I'll flag if we drift."
2. If the goal is vague (e.g. "work on cameras"), push back once: "Can you be more specific? What exact outcome should be committed by the end?"
3. Do not begin any work until the goal is confirmed.

Hold this goal in mind throughout the session. If the conversation starts drifting to a different topic, say: "This is drifting from our stated goal ([goal]). Should we finish that first, or are we intentionally switching?"
